# 홈 탭 뉴스 리스트 썸네일 정렬 수정

## 변경 내용

- 홈 탭의 뉴스 리스트 행 정렬을 상단 기준에서 중앙 기준으로 변경했다.
- 텍스트 영역 높이를 썸네일 높이에 맞춰 고정해, 제목 길이와 무관하게 썸네일과 본문이 균형 있게 보이도록 조정했다.
- 메타 정보는 카드 하단으로 정렬해 리스트 리듬을 일정하게 유지했다.
- 기본 이미지와 실제 썸네일 모두 `object-center` 기준으로 맞춰 어색한 크롭을 줄였다.
- 썸네일은 기존 대비 시각 면적 기준 약 2.5배 수준으로 확대해 홈 탭에서 이미지 존재감이 더 크게 보이도록 조정했다.
- 큰 썸네일 높이에 맞춰 제목과 요약 줄 수를 늘리고 본문 영역을 `space-between`으로 재배치해 세로 공백이 덜 비어 보이도록 조정했다.
- 맞춤 뉴스 리스트에는 별도 밀도 규칙을 적용해 썸네일을 줄이고 요약 줄 수를 늘려, 각 뉴스가 요약만으로도 빠르게 이해되도록 구성했다.
