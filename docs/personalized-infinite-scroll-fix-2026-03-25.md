# 맞춤 뉴스 무한 스크롤 흔들림 수정

## 배경

- 맞춤 뉴스 탭은 새 페이지를 불러올 때 전체 후보를 다시 개인화 점수로 정렬했다.
- 이 때문에 이미 화면에 보이던 카드의 위치가 바뀌면서 리스트가 살짝 흔들려 보였다.

## 수정 내용

- 화면에 한 번 노출한 맞춤 뉴스 카드 순서를 별도 상태로 유지한다.
- 다음 페이지 데이터가 들어오면 기존 카드 순서는 유지하고, 새 기사만 뒤에 이어 붙인다.
- 하단 로딩 영역 높이를 고정해서 스피너 표시 여부에 따른 작은 높이 변동도 줄였다.

## 영향 범위

- `my-news-front/app/page.tsx`
- 맞춤 뉴스 탭의 무한 스크롤 렌더링
