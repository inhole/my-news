# 개인화 구조

## 개요

`My News`는 로그인 없이도 최근 클릭 이력과 관심 카테고리를 바탕으로 맞춤형 뉴스 정렬을 제공합니다.
개인화 데이터는 서버 계정이 아니라 브라우저 로컬 저장소에 보관합니다.

## 핵심 원칙

- 개인화 데이터는 `localStorage`에만 저장합니다.
- 서버는 뉴스 원문, 카테고리, 날씨 데이터를 제공합니다.
- 클라이언트는 사용자 반응을 기반으로 개인화 점수를 계산합니다.
- 별도의 AI 요약 의존성 없이 설명문과 본문 데이터를 활용합니다.

## 저장 데이터

- `anonymousProfile.id`
- `categoryScores`
- `keywordScores`
- `preferredCategorySlugs`
- `preferredKeywords`
- `seenNewsIds`
- `createdAt`
- `updatedAt`

## 점수 계산

```text
personalizedScore = categoryAffinity * 2 + keywordAffinity + freshness - seenPenalty
```

- `categoryAffinity`: 선호 카테고리 반영 점수
- `keywordAffinity`: 기사 제목/설명 키워드 반영 점수
- `freshness`: 최신 기사 가산점
- `seenPenalty`: 이미 본 기사 감점
