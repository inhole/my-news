# 기능 정리

## 홈

- 오늘 주요 뉴스 헤드라인 제공
- 기사 제목/설명 기반 트렌딩 키워드 집계
- 최근 반응 기반 맞춤 뉴스 정렬
- 상단에서 날씨, 헤드라인, 트렌딩, 맞춤 탭 전환

## 뉴스

- 카테고리별 뉴스 목록 조회
- 검색어 기반 뉴스 검색
- 무한 스크롤 목록 탐색
- 뉴스 상세 페이지에서 본문, 이미지, 출처, 발행일 확인

## 개인화

- 로그인 없이 브라우저 로컬 저장소 기반 개인화
- 기사 클릭 시 카테고리/키워드 관심도 반영
- 마이페이지에서 관심 카테고리와 관심 키워드 확인
- 이미 본 기사 감점 처리

## 날씨

- 현재 위치 기반 날씨 조회
- 시간별/주간 날씨 제공
- 미세먼지와 대기질 정보 제공
- 위치 권한이 없으면 기본 위치로 대체

## 백엔드

- Naver 뉴스 API 수집
- 기사 메타데이터 및 본문 크롤링
- Neon PostgreSQL + Prisma 기반 저장
- 정기 뉴스 배치 실행
- `/health` 헬스체크 제공
