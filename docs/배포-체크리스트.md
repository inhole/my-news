# 배포 체크리스트

## 환경 변수

- 프론트와 백엔드가 운영 환경값을 사용하고 있는지 확인
- `DATABASE_URL`이 Neon Direct URL인지 확인
- `CORS_ORIGIN`이 실제 프론트 도메인인지 확인
- `ENABLE_SWAGGER=false`인지 확인
- `ENABLE_STARTUP_NEWS_FETCH=false`인지 확인
- `NAVER_CLIENT_ID`, `NAVER_CLIENT_SECRET`이 운영 키인지 확인

## 빌드 및 실행

- `npm run lint`
- `npm run test --workspace my-news-back -- --runInBand`
- `npm run build`
- `npm run start:prod`

## 기능 확인

- `/health` 응답 확인
- 홈, 뉴스 목록, 뉴스 상세, 마이페이지, 날씨 화면 확인
- 프론트 `/api` 리라이트 정상 동작 확인
- 배치 로그와 Naver API 오류 여부 확인
